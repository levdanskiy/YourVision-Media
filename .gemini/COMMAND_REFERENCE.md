# ⌨️ YOURVISION SIMPLE COMMANDS (YV-OS)
**Status:** ACTIVE | **Version:** 2.1 (Calendar Update)

Чтобы активировать новые команды, выполните:
`source .gemini/yv_aliases.sh`

---

## 🚀 1. УПРАВЛЕНИЕ КОНТЕНТОМ
*   **`yv_pub <файл>`**
    *Опубликовать пост (с проверкой длины и обновлением планов).*
    
*   **`yv_plan DD.MM.YYYY`**
    *Создать план на день (на основе базы событий).*
    
*   **`yv_status`**
    *Показать текущее состояние системы (долги, выполненные задачи).*

## 📅 2. КАЛЕНДАРИ И АФИША (NEW)
*   **`yv_guide`**
    *Сгенерировать пост **#LIVE_GUIDE** на СЕГОДНЯ (Ссылки на трансляции).*
    
*   **`yv_week`**
*   **`yv_history`**
    *Показать события "В этот день" из истории Евровидения.*
    *Сгенерировать пост **#WEEKLY_ROADMAP** (План на неделю).*

## 📡 3. РАЗВЕДКА И ДАННЫЕ
*   **`yv_time`**
    *Точное время (Рига).*
    
*   **`yv_odds`**
    *Показать текущие ставки букмекеров.*
    
*   **`yv_charts`**
    *Показать тренды стриминга.*

## ☁️ 4. СОХРАНЕНИЕ (GIT)
*   **`yv_sync`**
    *Быстрое сохранение: Добавить всё -> Коммит -> Отправить в облако.*
    
*   **`yv_save "Текст"`**
    *Сохранить локально с вашим комментарием.*

## 🛠️ 5. СИСТЕМА

*   **`yv_clean`**

    *Удалить временные файлы.*

    

*   **`yv_help`**

    *Показать эту справку.*



---



## 💻 6. БАЗОВЫЕ КОМАНДЫ (SHELL)

*   **`git commit -m "Сообщение"`**

    *Сохранить текущие изменения в истории (локально).*

    

*   **`git status`**

    *Показать, какие файлы изменены или готовы к сохранению.*

    

*   **`git push`**

    *Отправить сохраненные изменения в облако (GitHub).*

    

*   **`ls` / `ls -R`**

    *Показать файлы в текущей папке / во всех папках.*

    

*   **`cd <папка>`**

    *Перейти в другую папку (например, `cd .gemini/CONTENT`).*
