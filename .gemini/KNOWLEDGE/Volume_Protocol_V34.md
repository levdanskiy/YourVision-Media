# 📏 VOLUME PROTOCOL V35.0 (HYPER-GRADATION)
**// Стандарт длины контента для всех каналов Almanac //**

### 🧩 МАТРИЦА ГРЕЙДОВ (С ПРОБЕЛАМИ)

| Код | Название | Лимит (zn) | Назначение |
| :--- | :--- | :--- | :--- |
| **Q** | **QUOTE** | 100 - 300 | Цитата дня, афоризм, подпись к фото. |
| **S** | **SNAP** | 301 - 500 | Быстрый инсайд, факт, "Знаете ли вы". |
| **F** | **FLASH** | 501 - 700 | Срочная новость, молния. |
| **B** | **BRIEF** | 701 - 1000 | Утренний дайджест, список, рецепт (мини). |
| **N** | **NOTE** | 1001 - 1300 | Стандартный пост, размышление. |
| **C** | **COLUMN** | 1301 - 1800 | Авторская колонка, мнение. |
| **A** | **ARTICLE** | 1801 - 2400 | Аналитика, исторический обзор. |
| **R** | **REPORT** | 2401 - 3000 | Репортаж с места событий, тревел-лог. |
| **E** | **ESSAY** | 3001 - 4000 | Философское эссе, глубокий анализ. |
| **M** | **MONOGRAPH** | 4001 - 6000 | Фундаментальное исследование. |

### 🔄 ПРАВИЛА РОТАЦИИ
1.  **Запрет повтора:** Нельзя использовать один и тот же Grade два раза подряд на одном канале.
2.  **Контраст:** После "тяжелых" (R, E, M) обязательно должен идти "легкий" (Q, S, F, B).
3.  **Привязка к рубрикам:**
    *   #GLOBAL_EVENT_RADAR -> Обычно **N** или **C**.
    *   #SWEET_DREAM -> Обычно **F** или **B**.
    *   #HISTORY_MYSTERY -> Обычно **A** или **R**.

### ⚙️ ТЕХНИЧЕСКИЙ КОНТРОЛЬ
Скрипт `publish_tool.py` использует словарь `LIMITS` для жесткой валидации. Отклонение даже на 1 знак приведет к ошибке `REJECTED`.
