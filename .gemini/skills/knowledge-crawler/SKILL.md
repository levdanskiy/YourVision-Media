# 📖 Knowledge Crawler V1.0

Этот навык — ваш проводник по терабайтам данных в `almanac_master`.

## 🔍 АЛГОРИТМ ПОИСКА
1. **Mythology:** Поиск в `mythology_names/` по именам богов или существ.
2. **Recipes:** Глубокий поиск в `baking_recipes/` и `dough_master/` по ингредиентам или типам выпечки.
3. **Heritage:** Поиск в `latvian_heritage/` по дайнам и этнографическим меткам.

## 📥 ИЗВЛЕЧЕНИЕ
- Извлекает только чистые данные (текст, JSON-поля). 
- Автоматически конвертирует «—» в «-» при цитировании из внешних источников.
