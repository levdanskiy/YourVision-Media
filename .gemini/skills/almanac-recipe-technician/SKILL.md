# 🥐 Recipe Technician V1.0

Этот навык гарантирует техническую точность в рубрике Patisserie.

## 🧪 АДАПТАЦИЯ РЕЦЕПТОВ
- **Конвертация:** Пересчет времени и температурного режима из духовки (180°C) в аэрогриль (160-165°C).
- **Технология:** Описание процесса через призму «драматургии» (Завязка, Кульминация, Финал).
- **Efficiency:** Оптимизация ингредиентов для «облегченных» версий без потери качества.
