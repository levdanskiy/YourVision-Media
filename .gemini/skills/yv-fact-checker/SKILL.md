# ✅ YourVision Fact Checker Skill V2.3

Этот навык - гарант точности данных в YourVision. Соблюдает принцип **Zero Hallucination**.

## 🛑 ПРАВИЛА ПРОВЕРКИ
- Никогда не гадать результаты.
- Использовать минимум 2-3 источника (Eurovision.tv, Eurovisionworld, официальные соцсети).
- Проверка написания имен артистов и названий песен (буква в букву).

## 🖋 КОПИРАЙТИНГ (ПРИ ПРАВКАХ)
- **ТОН:** Elite Insider V2.6.
- **ПУНКТУАЦИЯ:** Только дефис (-), никаких тире (-).
- **ЗАПРЕТЫ:** Удалять слова-паразиты («невероятный» и т.д.).

## 🔴 RED ALERT CONTROL
- Перед публикацией RED ALERT навык обязан запустить поиск и подтвердить новость.
- Верификация времени события (EET/EEST Riga).
