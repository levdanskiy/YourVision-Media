# 📅 MASTER PROMPT: GRID GENERATOR
**// Протокол V12.0 Smart Grid //**

`СОЗДАЙ ПЛАН НА [DD.MM.YYYY]`

*Алгоритм сборки:*
1. Считать все мастер-планы (AC, NB, SW, YV).
2. Сверить с Live-календарем ESC и Глобальными событиями.
3. Разделить на блоки:
   - 🎯 СЕГОДНЯ (Только Ожидание).
   - 🚨 ДОЛГИ (Пропущенные слоты).
   - 🗓️ ПЛАН НА ДЕНЬ (С группировкой по каналам).
4. Сохранить историю ✅ выполненных в этом файле.
5. Использовать время по Риге (+1 CET).

### 🛠️ УЛУЧШЕННАЯ ЛОГИКА (V12.0):

**1. Системные Приоритеты:**
- **08:00 AC:** Обязательный слот **#GLOBAL_EVENT_RADAR** (Полный дайджест праздников).
- **Синхронизация:** Все остальные посты дня ОБЯЗАНЫ содержать контекстное упоминание праздников из Радара.

**2. Контроль Ротации (Mandatory):**
- **Оракулы:** Чередовать системы (Таро -> Руны -> Зодиак -> Оракулы -> Приметы). Не допускать повторов в течение одного дня для одного канала.
- **Модели (AC/NB):** Строгое чередование полов (Male -> Female -> Male). 
- **Aspect Ratio (AR):** Соблюдать правило 5-шаговой изоляции. Не использовать один и тот же AR дважды подряд.

**3. Управление Объемом (Grades):**
- Назначать грейд (S, A, B, C, D) для каждого слота в зависимости от важности темы.
- **S (3500)** - Юбилеи, спецпроекты.
- **A (2800)** - Глубокие легенды, аналитика.
- **B (2000)** - Стандарт.
- **C (1500)** - Инсайты.
- **D (500-1000)** - Новости, приметы.

**4. Разделение Брендинга:**
- **YourVision:** Для музыкальных/ESC постов.
- **Almanac:** Для AC, NB, SW.

**5. Интеграция Данных:**
- Использовать `Data Miner`, `Myth Finder` и `NASA RSS` для наполнения слотов фактурой перед генерацией.
