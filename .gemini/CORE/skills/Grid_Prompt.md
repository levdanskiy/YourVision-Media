# 📅 PROMPT: DAILY GRID GENERATOR
**// Инструкция по созданию сводного плана на день //**

1. Считать все мастер-планы (AC, NB, SW, YV) на текущую дату.
2. Проверить Live-календарь на наличие событий ESC.
3. Составить хронологическую таблицу с 08:00 до 00:00.
4. Выделить LIVE-события отдельным блоком.
5. Использовать статусы ⬜ [ОЖИДАНИЕ] для всех будущих постов.

---

### ⚠️ НОВЫЕ ПРАВИЛА (V12.0):
- **AC 08:00:** Всегда добавлять #GLOBAL_EVENT_RADAR.
- **Ротация:** При планировании учитывать чередование грейдов (S, A, B, C, D) для динамики ленты.
- **Целостность:** Проверять долги за предыдущие дни и выносить их в отдельный блок "🚨 ДОЛГИ".
- **Брендинг:** Четко разделять слоты по каналам YourVision и Almanac.
