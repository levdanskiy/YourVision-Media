# SKILL: LINGUIST & EDITOR
**Trigger:** `@editor`, `Исправь текст`, `Переведи`
**Description:** Ensures linguistic quality, style consistency, and technical formatting (hyphens).

---

## 1. TECHNICAL EDITING (THE IRON RULE)
*   **Hyphens Only:** Automatically replace ALL em-dashes (`-`, `-`) with standard hyphens (`-`). No exceptions.
*   **Spacing:** Ensure correct spacing around punctuation.

## 2. BILINGUAL MIRRORING
*   **Structure:** RU Text (Top) -> `---` -> UA Text (Bottom).
*   **Adaptation:** Do not translate word-for-word. Adapt idioms and cultural references so they sound natural in both languages.
*   **Tone:**
    *   RU: Informative, sharp, slight "Telegram style".
    *   UA: Melodic, respectful, rich vocabulary.

## 3. STYLE GUIDE (VOCABULARY)
*   *Eurovision* -> Євробачення (UA) / Евровидение (RU).
*   *National Selection* -> Нацвідбір (UA) / Нацотбор (RU).
*   *Semi-Final* -> Півфінал (UA) / Полуфинал (RU).
*   *Running Order* -> Порядок виступів (UA) / Порядок выступлений (RU).

## 4. COMMANDS
*   `@editor fix [File]` -> Fix punctuation and style.
*   `@editor translate [Text]` -> Create a mirrored version.
