# 🛡️ EXECUTION PROTOCOL: ABSOLUTE INTEGRITY (V1.2)
**// Высший стандарт исполнения и контроля качества //**

Каждая генерация ОБЯЗАНА проходить через следующие фильтры:

### 1. 🔍 DEEP DATA SCAN (100+)
- Запуск: `true_time`, `GOD_EYE`, `Oracle`, `Arcane`, `Intel_Hub`.
- Запрещено: Генерация без сверки с реальными событиями.

### 2. 🎞️ VISUAL ISOLATION (AR & MODELS)
- Ротация: 23 типа AR с 5-шаговой изоляцией (`ar_history.json`).
- Мандат: Модели (M/F) — только для AC и NB. Полный запрет людей для SW.

### 3. ✍️ TIER-1 COMPOSITION
- Язык: Текст — RU, Промпт — EN.
- Пунктуация: Только дефисы (-).
- Запрет: Никаких упоминаний нейросетей, AI, алгоритмов в тексте поста.

### 4. 🧪 CHANNEL SPECIFIC BLOCKS (DNA)
- **SWEET:** Обязательные блоки: "VISUAL PALETTE MAP", "ЛАБОРАТОРИЯ ИНГРЕДИЕНТОВ", "ТЕХНИЧЕСКИЙ ПРОТОКОЛ".
- **NEIGHBORS:** Обязательные блоки: "GPS Координаты", "Запахи", "#ORACLE_OMEN".
- **ARCHITECT:** Обязательные блоки: "Градусы планет", "Тектонический анализ".

### 5. ⚖️ POST VALIDATION (INQUISITION)
- Скрипт `post_validator.py` блокирует сохранение при нарушении Grade (A/B/C), наличии длинных тире или упоминании ИИ.
